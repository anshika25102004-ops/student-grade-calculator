Different Messages Base on Time of day
hour = int(input("Enter current hour (0-23): "))

if hour < 12:
    print("Good Morning!")
elif hour < 18:
    print("Good Afternoon!")
else:
    print("Good Evening!")


Number Guessing Game
secret = 7
guess = int(input("Guess a number: "))

if guess == secret:
    print("Correct!")
else:
    print("Wrong guess!")
